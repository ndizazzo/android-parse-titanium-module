enable

public static void enable(Context context)
Enable crash reporting for your app. This must be called before your application can use crash reporting.

The recommended way is to put a call to ParseCrashReporting.enable in your Application's onCreate method before Parse.initialize. For example: 

 public class MyApplication extends Application {
   public void onCreate() {
     ParseCrashReporting.enable(this);
     Parse.initialize(this, "your application id", "your client key");
   }
 }

Parameters:
context - The active Context for your application.
